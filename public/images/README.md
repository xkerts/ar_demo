# Product Images

Place product thumbnail images here with the following naming convention:
- armchair.jpg
- coffee-table.jpg
- floor-lamp.jpg
- bookshelf.jpg
- plant-pot.jpg
- office-desk.jpg

## Image Requirements
- Format: JPG, PNG, or WebP
- Recommended size: 800x600px (4:3 aspect ratio)
- Maximum file size: 500KB for optimal loading
- Use descriptive filenames matching product names

## Placeholder
If images are missing, the application will display a placeholder SVG automatically.

## Sources for Free Product Images
- Unsplash (https://unsplash.com)
- Pexels (https://www.pexels.com)
- Pixabay (https://pixabay.com)
